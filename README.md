Overview:

A stenotype keyboard is a specialized device used by stenographers and hobbyists that allows people to write fast enough, to keep up and write down human conversations. To be able to do that, it uses a special writing method known as shorthand, which focuses on the phonetics of each word, instead of how it is written.
This device was fully designed, programmed and created by me, as a part of my engineering thesis project. The full details of this work can be accessed by reading my paper which is assigned to this repository.
